## 1. 目标

为 `photo-outline-bin` 增加一套比“单独 L 标尺 + 任意桌面背景”更稳定的拍摄方案：

- 给用户一个可直接下载打印的 `A4` 校准底纸
- 让物体放在固定区域内，降低背景噪声
- 为后续透视校正和整页尺度标定预留识别标记
- 保留和当前 `L` 标尺识别流程的兼容性

## 2. 版式设计

纸张规格：

- 成品尺寸：`210 x 297 mm`，标准 `A4`
- 打印要求：`100% actual size`
- 建议纸张：普通哑光打印纸即可，尽量避免高反光相纸

版面结构：

- 四角放置低干扰定位角标
- 中央设置物体摆放区
- 左下角内嵌一枚与当前系统一致的 `80 x 60 mm` `L` 形标尺
- 底纸上只放低干扰辅助线，不在物体摆放区放深色文字或深色块

## 3. 为什么优于单独标尺

仅用单独标尺时，系统主要解决“局部尺度换算”，但仍然暴露在以下问题里：

- 背景纹理复杂，前景分割不稳定
- 手机有轻微俯仰，物体和标尺会出现透视形变
- 用户不知道该把物体放在画面哪里，拍摄一致性差

`A4` 背景纸可以额外提供：

- 固定拍摄平面
- 固定物体放置区域
- 更一致的浅色背景
- 后续可扩展为整页透视校正和裁切

## 4. 与当前实现的兼容方式

这次的底纸不是“等未来算法完成后才能用”，而是分两层工作：

### 4.1 当前可直接使用的兼容层

- 左下角内嵌当前同规格 `L` 形标尺
- 现有识别逻辑仍按 `L` 标尺工作
- 低干扰角标和浅色辅助线不会主动参与当前 `V1` 识别

也就是说，当前系统不会识别整张 `A4` 纸，但可以先识别底纸左下角的内嵌 `L` 标尺。

### 4.2 后续可升级的整页识别层

后续算法升级后，再利用四角标记进入真正的 `A4` 校准模式。

## 5. 建议的识别流程

### Phase 0：当前兼容模式

流程：

1. 用户下载并打印 `A4` 校准底纸
2. 把物体放进中央摆放区
3. 俯拍整张纸的一部分，至少保证左下角内嵌 `L` 标尺可见
4. 系统沿用当前 `L` 标尺识别流程

优点：

- 无需改动现有核心识别链路
- 可以立刻提升背景一致性

限制：

- 仍然主要依赖局部 `L` 标尺
- 还没有真正做透视矫正

### Phase 1：A4 标记识别模式

建议流程：

1. 识别四角标记
2. 建立 `A4` 平面的四点映射
3. 对原图做透视校正，拉正到标准 `210 x 297 mm` 平面
4. 按底纸上的物体摆放区裁切 ROI
5. 在 ROI 内做目标物轮廓提取
6. 若四角标记缺失，则回退到左下角 `L` 标尺模式

这一版的核心收益：

- `mm/px` 不再只依赖单个角上的标尺
- 轮廓提取范围可限定在摆放区内
- 对桌面背景和边缘干扰更稳

## 6. 标记设计建议

建议保留以下约束：

- 四角标记尺寸统一，便于做模板匹配
- 标记应放在纸面边缘内侧，避免被打印机裁切
- 标记颜色和底纸背景要有稳定反差
- 标记外形不要与被收纳物常见轮廓过于相似

当前交付的底纸采用：

- 四角低干扰定位角标
- 中央浅色物体放置区，不放深色提示字
- 左下角深色 `L` 标尺

## 7. 用户操作建议

建议界面文案按下面语义引导：

- 优先推荐使用 `A4` 校准底纸拍摄
- 纸张必须平整，不要卷边
- 打印时不要勾选“适应页面”
- 当前 `V1` 以左下角内嵌 `L` 标尺为准，不要求把四角角标全部拍进画面
- 物体尽量放在中央摆放区，不遮挡左下角 `L` 标尺

## 8. 前端改造建议

如果进入 `Phase 1`，建议新增这些展示项：

- 底纸识别状态：`未识别 / 部分识别 / 已识别`
- 透视校正预览图
- 裁切后的摆放区预览
- 当前缩放比例和纸面像素密度
- 回退原因：例如“角标遮挡，已回退到 L 标尺模式”

## 9. 风险与规避

风险：

- 用户打印缩放不正确
- 纸张翘起导致平面假设失效
- 角标被手或物体遮挡
- 纸张边缘落在强阴影里，影响标记识别

规避方式：

- 底纸上直接印 `Print at 100% actual size`
- 保留 `L` 标尺回退方案
- 审查区展示角标识别结果和回退状态
- UI 明确提示“纸面尽量铺平、保持俯拍”

## 10. 当前结论

如果目标是“综合体验更稳定”，`A4` 校准底纸是值得推进的方向。

建议的落地顺序：

1. 先把 `A4` 底纸下载入口提供给用户
2. 当前阶段沿用左下角内嵌 `L` 标尺保持兼容
3. 下一阶段再补四角标记识别、透视校正、摆放区裁切
